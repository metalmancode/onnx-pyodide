## Short post

I open-sourced the reproducible core of an ONNX-for-Pyodide port developed for
MLDeck.

The browser proof trains a scikit-learn model, converts it with skl2onnx,
validates and serializes it with ONNX Python, then executes it with ONNX Runtime
Web—all in one browser tab.
